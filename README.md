# 🏥 MediCare.Sys - Advanced Hospital Management System

A high-end, production-ready Full-stack Hospital Management System (HMS) built with the **MERN Stack** (MongoDB, Express, React, Node.js). This platform provides specialized dashboards for **Admins, Doctors, and Patients**, featuring a premium UI with glassmorphism, dynamic charts, and smooth animations.

---

## 🚀 Key Features

### 👑 Admin Management Dashboard
*   **Total Control:** Manage Doctors, Patients, and Departments.
*   **Financial Oversight:** Track billing, payments, and hospital income.
*   **Real-time Analytics:** Visualized stats for hospital performance using Chart.js.
*   **System Notifications:** Stay updated on new registrations and system events.
*   **Security:** Advanced profile management and secure password updates.

### 🩺 Professional Doctor Dashboard
*   **Appointment Management:** View, accept, and manage patient bookings.
*   **Prescription System:** Digital prescription creation and management.
*   **Patient Records:** Access detailed medical history for assigned patients.
*   **Analytics:** Track patient visits and monthly consultations.
*   **Profile Control:** Professional profile customization and secure access.

### 👤 Integrated Patient Portal
*   **Easy Booking:** Discover specialists and book appointments instantly.
*   **Digital Health Records:** Access prescriptions and lab reports (PDF download supported).
*   **Financial Tracking:** View invoices and track payment history.
*   **Profile Management:** Update personal information and emergency contacts.
*   **Instant Alerts:** Notifications for upcoming appointments and report uploads.

---

## 🛠️ Technology Stack

*   **Frontend:** React.js (Vite), Tailwind CSS, Lucide React (Icons), Chart.js.
*   **Backend:** Node.js, Express.js.
*   **Database:** MongoDB (Mongoose ODM).
*   **Authentication:** JWT (JSON Web Tokens) with Role-Based Access Control (RBAC).
*   **Styling:** Modern UI/UX with Glassmorphism and Responsive Design.

## 💻 Getting Started

### Prerequisites
*   Node.js (v16.x or higher)
*   MongoDB Atlas account or local MongoDB instance

### 1. Project Setup
```bash
# Clone the repository
git clone https://github.com/JayaniSrimali/Hospital-Management-System.git
cd Hospital-Management-System
```

### 2. Backend Configuration
```bash
cd server
npm install
# Create a .env file and add your MONGODB_URI and JWT_SECRET
npm start
```

### 3. Frontend Configuration
```bash
cd client
npm install
npm run dev
```
